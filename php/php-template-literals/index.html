PHP 與 TypeScript 的 Template Literals

PHP 也有支援如 TypeScript 的 Template Literal
Contents
  1. Version
  2. TypeScript
  3. PHP
  4. Conclusion

TypeScript 提供了 Template Literals，可以讓字串處理的可讀性更高，不再只是湊字串而已，事實上，PHP 也有這種機制。

Version

PHP 7.0.0
TypeScript 1.8.9

TypeScript

在 TypeScript，只要字串改用反引號 ，再配合 ${} 即可。

PHP

在 PHP，只要字串改用雙引號，再配合{$}即可。

剛好與 TypeScript 相反。

Conclusion

  • Template literals 方式湊字串，遠比傳統用 + 或 . 的方式可讀性高，最少可以先看到字串整體的全貌。
