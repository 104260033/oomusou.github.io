如何根據 Checkbox 選取與否決定 Button 的 Enable?

以 Data Driven 的方式開發 Vue.js
Contents
  1. Version
  2. 單一Checkbox
  3. 多個 Checkbox
  4. Conclusion
  5. Sample Code

實務上 GUI 常需要做一些防呆，如勾選了 I agree 的之後，button 才能 enable，否則為 disable；又如若完全沒有選擇任何產品時，button 為 disable，只要選擇了一個以上的產品，button 為 enable，這在 Vue.js 是否有不同的作法呢?

Version

Vue.js 1.0.18

單一Checkbox

See the Pen Vue.js Checkbox and Enable Button by oomusou (@oomusou) on CodePen.

預設 button 為 disable，當勾選了 I Agree 之後，button 變成 enable。

See the Pen Vue.js Checkbox and Enable Button by oomusou (@oomusou) on CodePen.

HTML 很簡單，只有 checkbox 與 button。

要給 Vue.js 管理的 <div>，要加上 id。

checkbox 使用了 v-model 與 checked 做 data binding，v-model 為 Vue.js 自己的 directive。1 1凡 v- 開頭的屬性皆為 Vue.js 所提供的 directive。

button 直接將 disable 屬性與 checked 做 data binding，:為 Vue.js 所擴充的符號，表示將 HTML 屬性做 data binding。

  • 當勾選了 checkbox 時，checked 為 1，也就是 true，加了 ! 為 false，為 disabled=false，也就是 enable。
  • 當不勾選 checkbox 時，checked 為 0，也就是 false，加了 ! 為true，為 disabled=true，也就是 disable。

See the Pen Vue.js Checkbox and Enable Button by oomusou (@oomusou) on CodePen.

el 指定 Vue.js 所掌管的 <div> 區塊的 id。

data 列出了 checked 變數，也就是要做 data binding 的變數。

多個 Checkbox

See the Pen Vue.js Checkboxs and Enable Button by oomusou (@oomusou) on CodePen.

預設 button 為 disable，當勾選任意 checkbox 後，button 變成 enable，若沒勾選任 checkbox，則 button 又變為 disable。

See the Pen Vue.js Checkboxs and Enable Button by oomusou (@oomusou) on CodePen.

HTML 很簡單，只有多個 checkbox 與 button。

3 個 checkbox 的 v-model 都與相同的 checked 做 data binding，不過此時 checked 為陣列。

button 直接將 disable 屬性與 checked 做 data binding。

  • 當勾選了多個 checkbox 時，checked 陣列的 length 大於 0，為 true，加了 ! 為 false，為 disabled=false，也就是 enable。
  • 當沒勾選任何 checkbox 時，checked 陣列的 length 等於 0，為 false，加了 ! 為 true，為 disabled=true，也就是 disable。

See the Pen Vue.js Checkboxs and Enable Button by oomusou (@oomusou) on CodePen.

JavaScript 非常精簡，與單一 checkbox 的差別只有 false 改成 []，完全不用寫任何邏輯判斷，也不用處理任何 event。

Conclusion

  • Vue.js 的思維與 jQuery 不同，寫 jQuery 需要思考的是 event，但寫 Vue.js 需思考的是 data，透過 data binding，我們可以將邏輯放在處理 data，而不是如 jQuery 是在處理 DOM，因此 Vue.js 的程式非常精簡。
