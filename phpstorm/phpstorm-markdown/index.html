如何使PhpStorm支援Markdown?

只要安裝Markdown Support Plugin就可以在PhpStorm編輯Markdown
Contents
  1. Version
  2. 安裝Plugin
  3. Conclusion

由於Github的流行，越來越多人喜歡用markdown格式寫文件。使用Hexo寫blog時，我會使用Sublime Text寫markdown，但若寫專案的readme.md時，由於是在PhpStorm內工作，所以會希望也能在PhpStorm內寫markdown。可惜PhpStorm內建並沒有支援markdown格式，所幸JetBrains有另外開發plugin，讓我們也可以在PhpStorm寫markdown了。

Version

PhpStorm 9.0.2
Markdown Support Plugin 8.0.0.20150929

安裝Plugin

PhpStorm -> Preference -> Plugins -> Install JetBrains plugin…

輸入 markdown，找到 Markdown support。

安裝完會要求重新啟動PhpStorm。

以後只要開啟*.md格式，就會出現左右分隔畫面，其中左側為我們編輯markdown的文字編輯器，右邊為所見即所得，讓我們不用存檔就可以立刻看到render結果，非常方便。

Conclusion

  • Markdown格式非常簡單，而且所寫即所得，寫熟了之後非常有效率，建議大家文件使用markdown格式撰寫，將來要放在Hexo，或轉成HTML或PDF格式都很方便。
