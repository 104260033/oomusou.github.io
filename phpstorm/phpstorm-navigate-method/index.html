如何快速在PhpStorm切換method?

分享兩個快速切換method的方式
Contents
  1. Version
  2. 列出method清單
  3. 快速切換上下method
  4. Conclusion

PhpStorm是很方便的PHP IDE，除了測試、重構與偵錯很方便外，還提供了各式各樣的快捷鍵可用，本文分享2個快速切換method的方法。

Version

PhpStorm 10.0.2

列出method清單

熱鍵 : ⌘ + F12

會列出目前class所有的method清單，可以直接上下鍵選取。

也可以直接輸入method名稱做過濾。

快速切換上下method

熱鍵 : ⌃ + ↑ 上一個method
熱鍵 : ⌃ + ↓ 下一個method

但這兩個熱鍵，預設會與OS X的Mission Control與Application Windows的熱鍵相衝，必須自行停用。

有了這兩個熱鍵，就不一定需要觸控板或滑鼠的滾輪來快速捲動了。

Conclusion

  • PhpStorm還有很多好用的熱鍵，但因為這兩個熱鍵寫code最常用，且會與OS X預設熱鍵相衝，所以特別紀錄下來。
