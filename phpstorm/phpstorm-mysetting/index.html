PhpStorm預設的設定已經非常好用，但有一些進階的功能，預設並沒有開啟，解除這些封印後，將全面釋放PhpStorm的淺力。

Version

PhpStorm 2016.1

主題

第一次執行PhpStorm，會要求你設定熱鍵/IDE/顏色字型主題。

個人偏好黑底白字的風格，所以IDE與顏色字型都選擇Darcula。

字型大小

PhpStorm的字型大小預設是12，個人覺得稍微小了一點，會將字型稍微調大。

Editor

PhpStorm -> Preferences -> Editor -> Color & Fonts -> Font

我會將editor字型調成15。

IDE

PhpStorm -> Preferences -> Appearance & Behavior -> Appearance

勾選Override default fonts by (not recommend)，將IDE字型改成16。

Terminal

PhpStorm -> Preferences -> Editor -> Colors & Fonts -> Console Font

在PhpStorm內可以直接在terminal下指令，將console字型改成16。

Whitespace

顯示Whitespace

PhpStorm -> Preferences -> Editor -> General -> Appearance

  • 將Show whitespaces與其下的Leading、Inner、與Trailing打勾。1 1詳細請參考如何讓PhpStorm在存檔時自動刪除whitespace?

存檔自動刪除Whitespace

PhpStorm -> Preferences -> Editor -> General

  • 將Allow placement of caret after end of line打勾。
  • 將Strip trailing spaces on Save選All。2 2詳細請參考如何讓PhpStorm在存檔時自動刪除whitespace?
